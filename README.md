# MoneyPilot 💰

MoneyPilot is a full-stack personal finance tracking web application designed to help users monitor their expenses, understand spending patterns, and manage their monthly budgets effectively.

The application provides a clean dashboard, interactive analytics, and simple expense management features to make personal finance tracking easy and intuitive.

---

## Features

- User Authentication (Register / Login)
- Add, edit, and delete expenses
- Expense categorization
- Dashboard with financial summary
- Category-wise expense analysis
- Monthly spending visualization using charts
- Budget tracking and insights
- Light and Dark theme toggle
- Responsive design for mobile and desktop
